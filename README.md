# DiscordBot-Webpage-Status
A Discord bot that monitors the online status of websites, displays their metadata, and updates their status periodically using embedded messages.

All website datra is stored in a SQL database with Message ID and Channel ID. The Bot will periodicaly scape the databse and vailidate if the website is accessable or not accessable and update the status message.
